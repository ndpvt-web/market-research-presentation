## Industry Analysis

```
SLIDE 1:  [Cover]
  Layout: title_slide
  Content: Title, date, "Prepared by [User/Team]", disclaimer line

SLIDE 2:  [Executive Summary]
  Layout: exec_summary
  Content: 3-5 bullet key findings, key metrics sidebar (market size, CAGR, # players)
  Action title: One-sentence governing thesis

SLIDE 3:  [Section Divider] Market Overview
  Layout: section_divider

SLIDE 4:  [Market Size & Growth]
  Layout: full_chart
  Chart: Bar chart (historical) + line (forecast CAGR)
  Data: Market size by year (5yr history + 3-5yr forecast)
  Source: Industry report citation

SLIDE 5:  [Market Segmentation]
  Layout: chart_with_text
  Chart: Stacked bar or pie showing segment breakdown
  Text: Key segment descriptions and growth drivers
  Source: Industry report

SLIDE 6:  [TAM/SAM/SOM]
  Layout: framework_2x2
  Visual: Nested circles with methodology callouts
  Data: Dollar values for each tier + growth rates

SLIDE 7:  [Section Divider] Industry Dynamics
  Layout: section_divider

SLIDE 8:  [Porter's Five Forces]
  Layout: full_chart
  Chart: Radar/pentagon chart with 5 axes
  Callouts: Score and key driver for each force

SLIDE 9:  [PESTEL Overview]
  Layout: comparison_table
  Table: 6 factors x (Finding | Impact | Trend)
  Color-coding by impact level

SLIDE 10: [Key Industry Trends]
  Layout: three_columns
  Content: Top 3 trends with data points and implications

SLIDE 11: [Section Divider] Competitive Landscape
  Layout: section_divider

SLIDE 12: [Market Share Analysis]
  Layout: full_chart
  Chart: Horizontal bar chart or pie showing share by company
  Source: Market research firm

SLIDE 13: [Competitive Positioning]
  Layout: full_chart
  Chart: Bubble/scatter plot (2 axes + bubble size = revenue)
  Labels: Company names on each bubble

SLIDE 14: [Peer Comparison Table]
  Layout: comparison_table
  Table: Companies x metrics (Revenue, Growth, Margin, Valuation)
  Highlight: Best/worst in each column

SLIDE 15: [Section Divider] Financial Analysis
  Layout: section_divider

SLIDE 16: [Revenue & Growth Trends]
  Layout: chart_with_text
  Chart: Line chart showing revenue trajectories of top 3-5 players
  Text: Growth rate comparison and inflection points

SLIDE 17: [Profitability Analysis]
  Layout: full_chart
  Chart: Grouped bar comparing margins across peers
  Callout: Industry average line

SLIDE 18: [Section Divider] Outlook & Risks
  Layout: section_divider

SLIDE 19: [SWOT Analysis]
  Layout: framework_2x2
  Visual: 4-quadrant with 3-5 data-backed items each

SLIDE 20: [Risk Matrix]
  Layout: full_chart
  Chart: 3x3 scatter with labeled risk points
  Color zones: Green/Yellow/Red

SLIDE 21: [Key Catalysts & Timeline]
  Layout: timeline
  Visual: Horizontal timeline with 4-6 dated catalyst events

SLIDE 22: [Scenario Analysis]
  Layout: comparison_table
  Table: Bear/Base/Bull x key metrics
  Probability weights for each scenario

SLIDE 23: [Recommendations]
  Layout: exec_summary
  Content: 3-5 actionable recommendations with supporting data

SLIDE 24: [Appendix: Data Sources]
  Layout: appendix
  Content: Full bibliography with numbered sources

SLIDE 25: [Appendix: Methodology]
  Layout: appendix
  Content: Research methodology, data collection dates, limitations
```

## Layout Type Reference

| Layout | When to Use | Key Elements |
|---|---|---|
| `title_slide` | Cover only | Title, subtitle, date, disclaimer |
| `exec_summary` | Key findings, recommendations | 3-5 bullets + optional metrics sidebar |
| `section_divider` | Section breaks | Section title + number |
| `full_chart` | Data-heavy slides | Chart at 80% of slide, title, source line |
| `chart_with_text` | Chart + narrative | 60/40 split: chart + bullet commentary |
| `comparison_table` | Peer benchmarking, features | Formatted table with conditional formatting |
| `framework_2x2` | SWOT, BCG, positioning | 4-quadrant diagram |
| `three_columns` | 3 key points/trends | Icon/number + title + description x3 |
| `timeline` | Catalysts, milestones | Horizontal timeline with dated events |
| `appendix` | Supporting data, sources | Dense content, smaller font acceptable |
